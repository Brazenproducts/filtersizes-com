Nominal vs Actual Filter Size

The label on the box is usually rounded. The filter you hold in your hand is usually a bit smaller.

Home / Nominal vs Actual

What nominal size means

Nominal size is the name used to sell and sort filters. It is a simplified, rounded dimension such as 16x20x1. That is the size shoppers remember and the size retailers organize around.

What actual size means

Actual size is the true physical dimension of the filter frame. Manufacturers typically make the filter slightly smaller than the nominal label so it can slide into place without binding.

Example A filter sold as 16x20x1 may measure something closer to 15.5 x 19.5 x 0.75 inches. Exact numbers vary by brand and style, but the pattern is normal.

Why the difference exists

HVAC filter slots and return grilles need a little tolerance. If filters were made exactly to the rounded label, many would be frustratingly tight or impossible to install. Slightly reduced actual dimensions make replacement easier, but they also mean two products with the same nominal label may not fit identically.

When actual dimensions matter most

  • Older systems with imperfect filter racks
  • Media cabinets using deeper 4-inch or 5-inch filters
  • Homes where the current filter already feels sloppy or tight
  • Any replacement decision after a prior fit problem

How to shop smarter

Start with the nominal size because that is how the market is organized. Then check actual dimensions in the product listing whenever possible, especially for thicker filters and premium options. If a listing does not disclose actual dimensions and your system is sensitive to fit, move on.
